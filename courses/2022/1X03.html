Unofficial Practice Test for Midterm 2

There is no official practice test as there was for the first midterm. I have put together an unofficial practice test which you can use to review for the test. As with the official practice test for the first midterm, the practice test should not lure you into a false sense of security: only completing the attached problems is not sufficient studying for the midterm.

You can also review this list of recent tutorial problems. Some of these have been covered in tutorial, while some have not.

Tutorial Problems

Tutorials for this class are a time to ask questions: concepts you are unsure about, problems you are having difficulty with, etc. To take full advantage of tutorial, I recommend preparing questions for tutorial ahead of time by:

  • Making note of questions you have in lecture or while doing practice problems as they come up.
  • Reviewing content before tutorial for concepts or problems to ask about.

Questions to ask in tutorial can be about a concept, definition, theorem, a practice problem, or anything else you are unsure about.

You can ask questions live in tutorial or request them ahead of time via email, to cummim5@mcmaster.ca.
